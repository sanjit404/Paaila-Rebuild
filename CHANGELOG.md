# Changelog
All notable changes and improvements to Paaila are documented here.

## [Second Change] - 2026-04-05

### Major Changes - Improvements and Additions
This release represents a significant rewrite and enhancement of the application, including 20+ improvements and the addition of 5 major features.


### New Features (4+)

#### 1. New Feed System for dynamic content display
- User can access the feed from the navigation bar
- Displays dynamic and real-time content updates

#### 2. New Like and Interaction System
- Users can engage with posts through likes and ratings
- Improves overall interaction within the platform

#### 3. New Trending System to show what’s currently happening
- Displays highly engaged and popular posts
- Accessible from the feed section

#### 4. Traveler’s additional information added to the tracking page
- Tracker can view additional traveler data such as speed, coordinates, and online status
- Enhances live tracking visibility

#### 5. Map Layer Customization
- Users can switch between different map styles (Satellite, Street, Outdoor)
- Available in the map layer section of the tracking view
